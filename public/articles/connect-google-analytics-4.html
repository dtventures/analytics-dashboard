How to Connect Google Analytics 4 to Your Website

Google Analytics 4 is the current generation of Google's free web analytics platform. It replaced Universal Analytics in 2023. Here's how to create a GA4 property and add the tracking code to your site so you can start measuring real visitor data.

What is Google Analytics 4?

GA4 is Google's free analytics platform that measures what visitors do on your website. Once installed, you can see:

  • Sessions and users — how many people visit and how often they come back
  • Traffic sources — where visitors come from (Google, social, direct, referrals)
  • Top pages — which pages get the most views and keep people engaged longest
  • Audience data — device types, countries, new vs. returning visitors
  • Events and conversions — button clicks, form submissions, purchases
  • Realtime data — who's on your site right now
💡
Completely free. Google Analytics 4 is free for all websites. There's also a paid version (GA4 360) for enterprise, but the free tier is what everyone uses and it's more than enough for most sites.
⚠️
Universal Analytics is gone. If you still have the old UA tracking code (starting with UA-) on your site, it stopped collecting data in July 2023. You need GA4 (which uses a G- Measurement ID) to get current data.

Step-by-Step Setup

1

Go to Google Analytics

Visit analytics.google.com and sign in with your Google account. Use the same account you want to use for AnalyticsDash.

2

Create a New Account (if needed)

If this is your first time, click "Start measuring". If you already have an Analytics account, click the gear icon (Admin) in the bottom left, then click "Create" → "Account".

Give your account a name — usually your business or organization name, e.g. Lighthouse Launch. You can have multiple properties under one account, so name the account broadly.

3

Create a GA4 Property

A "property" in GA4 represents one website (or app). Give it a descriptive name like lighthouselaunch.com, select your reporting time zone and currency, then click Next.

You'll be asked about your business. Fill in your industry and business size — these only affect Google's default report recommendations, not the data you collect.

4

Set Up a Data Stream

A data stream is the source of data into your property. For a website, choose "Web". Enter your website URL and give the stream a name.

Make sure to enter your actual domain, e.g. https://lighthouselaunch.com. After saving, you'll land on the stream details page — this is where your Measurement ID lives (it looks like G-XXXXXXXXXX). Keep this page open — you'll need this ID in the next step.

ℹ️
Enhanced measurement is on by default. GA4 automatically tracks page views, scrolls, outbound clicks, site search, video engagement, and file downloads without any extra code. You can toggle these off per-stream if needed.
5

Install the Tracking Code

You need to add the Google tag to every page of your website. The method depends on how your site is built.

Method 1 — Direct HTML (recommended for custom sites)

Copy this snippet and paste it into the <head> section of every HTML page on your site. Replace G-XXXXXXXXXX with your actual Measurement ID:

<!-- Google tag (gtag.js) --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXXXX'); </script>

Method 2 — Google Tag Manager (GTM)

If you use Google Tag Manager, create a new tag of type Google Tag, enter your Measurement ID, set the trigger to All Pages, and publish. This is the preferred method for sites that already have GTM installed.

Method 3 — CMS plugins

  • WordPress: Use the Google Site Kit plugin — it installs GA4 with one click and also connects Search Console.
  • Webflow: Site Settings → Integrations → Google Analytics → paste your Measurement ID.
  • Squarespace: Settings → Advanced → External API Keys → paste your Measurement ID.
  • Shopify: Online Store → Preferences → Google Analytics → connect your account.
  • Wix: Marketing & SEO → Marketing Integrations → Google Analytics → connect.
6

Verify It's Working

Open Google Analytics, go to Reports → Realtime. Then open your website in a new tab. Within 30–60 seconds you should see yourself appear as an active user.

💡
Nothing showing? If you don't appear in realtime, check that: (1) the tag is actually in your page source — right-click → View Page Source and search for gtag, (2) you don't have an ad blocker active (they block GA), (3) you deployed the change (some site builders need you to publish after editing).

Understanding GA4 Key Concepts

GA4 works differently from the old Universal Analytics. A few things worth knowing:

  • Everything is an event. Page views, clicks, scrolls — all tracked as events. This is more flexible but takes some getting used to.
  • Sessions work differently. GA4 uses engagement-based sessions. "Bounce rate" in GA4 is the inverse of engagement rate — a bounce is a session with less than 10 seconds of engagement, no conversion, and no second page view.
  • Conversions are events marked as key events. Go to Admin → Events, find any event (like purchase or form_submit), and toggle "Mark as conversion" to start counting it.
  • Data takes 24–48 hours to appear in most reports. Realtime is instant. Standard reports show data from yesterday and earlier.
  • Audiences and explorations are powerful. Use Explore → Funnel Exploration or Segment Overlap for custom analysis beyond standard reports.

Setting Up Conversions (Optional but Important)

If you have goals — contact form submissions, newsletter signups, purchases — you should mark them as conversions. GA4 auto-tracks some events; others need manual setup.

Automatic events (tracked by default)

  • purchase — for e-commerce with enhanced e-commerce set up
  • first_visit — new visitor to your site
  • session_start — start of every session

Track contact form submissions

Add a small snippet to your form's success handler:

gtag('event', 'form_submit', { 'event_category': 'Contact', 'event_label': 'Contact Form' });

Then in GA4: Admin → Events → find form_submit → toggle "Mark as key event".

ℹ️
Thank-you page method. If your form redirects to a thank-you page (e.g. /thank-you), you can create a conversion by going to Admin → Conversions → New conversion event and entering the event name that fires on that page. This is often simpler than adding custom JavaScript.
