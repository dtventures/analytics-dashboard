Sign in with Google
Only you can access this dashboard. OAuth tokens are stored in your server session.

First time? Quick setup needed:

Go to console.cloud.google.com → New Project
Enable: Analytics Data API, Analytics Admin API, Search Console API
OAuth Consent Screen → External → add your email as test user
Create OAuth Client ID → Web app → add redirect URI: http://localhost:3000/auth/callback
Copy Client ID + Secret into your .env file and restart the server
